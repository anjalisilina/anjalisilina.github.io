My Projects

Here's an overview of the technical projects I've worked on. My work spans AI/ML systems, full-stack development, data visualization, and cybersecurity tooling. Each project has pushed me to apply both my CS and Finance backgrounds in creative ways.

FinAgent is my most comprehensive project, a multi-agent RAG system built with Azure AI Search and OpenAI LLMs. It ingests earnings call transcripts, performs sentiment analysis, and surfaces structured financial insights. It's designed to make financial research faster and more accessible.

During my time at Code4Community, I built and maintained features for a platform connecting nonprofits with volunteers. This gave me deep experience with collaborative Git workflows, code reviews, and building for real users with real needs.

At NETSCOUT, I worked on cybersecurity tools for network traffic analysis and threat detection, a completely different domain that sharpened my understanding of infrastructure and security at scale. I also worked on auditing a large database of VPNs for an acquired company, and cleaned the dataset using Excel.

I've also worked on several academic projects including reinforcement learning agents for CS 4100, data visualization dashboards using Altair and Airbnb data, and ML models for classification and regression tasks. These projects are at the intersection of theory and practical application.
